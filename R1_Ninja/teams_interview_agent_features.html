Teams-integrated interview agent with proctoring — full feature list

Teams integration layer Microsoft Teams

Bot presence in call
  • AI agent joins as a bot participant via Teams Calling APIs / Graph API
  • Speaks questions aloud using Azure Cognitive Speech TTS
  • Listens to and transcribes candidate answers in real time (Azure STT)
  • Sidebar panel shows the recruiter a live transcript + flag feed
Screen share analysis
  • Agent requests and receives candidate's screen share stream
  • Frame-by-frame OCR detects open tabs, browser URL bar, IDE content
  • App classifier flags: ChatGPT, Copilot, StackOverflow, GitHub Copilot
  • Candidate warned in-call if disallowed app or URL is detected

Proctoring signals Anti-cheat

Eye tracking
  • WebGazer.js / MediaPipe FaceMesh tracks gaze via webcam feed
  • Gaze zone map: screen-centre (safe), left/right/down (flag)
  • Sustained off-screen gaze (>3 s) triggers a gentle verbal nudge
  • Repeated off-gaze events logged as integrity flags in report
Face & presence checks
  • Face count detection — flags if more than one person is in frame
  • Face absence alert after 5 s (candidate left desk)
  • Identity re-verification snapshot at random intervals (vs. ID on file)
  • Lip-sync check: mic audio must correlate with mouth movement
Audio environment checks
  • Background voice detection — flags whispered coaching
  • Earphone / headset detection heuristic (mono vs. stereo echo pattern)
  • Audio gap anomaly — long silences followed by fluent answers flagged
  • Keystroke-while-speaking detection (screen share audio layer)
Navigation & tab control
  • URL whitelist enforced — only approved domains reachable during test
  • Agent narrates a warning when a blocked link is clicked on screen
  • New window / tab open event captured and flagged instantly
  • Alt+Tab / Cmd+Tab frequency logged as a distraction signal
Answer integrity checks
  • Copy-paste detection in code editor (clipboard diff vs. typed)
  • AI-generated text classifier on verbal + written answers
  • Response time vs. question complexity — suspiciously fast flags
  • Vocabulary drift analysis — inconsistent register across rounds
Real-time candidate guidance
  • Agent speaks a verbal warning on first violation — not silent
  • Second violation triggers an on-screen banner + verbal reminder
  • Third violation escalates to recruiter with option to end session
  • All warnings timestamped in the integrity log
